Есть программа, которая сжимает фотографии
Есть папка «Source» с фотографиями
Каждую фотографию мы хотим уменьшить до 200px в ширину ( высота меняется пропорцион ально)
Нужно для каждой фотографии запустить программу и результат работы положить в папку «Result»
Дополнительное задание: запускать 4 процесса параллельно и разделить фотографии между ними

Для домашней работы используйте Image Magick для Windows и Linux, либо встроенную утилиту sips для mac
http://www.imagemagick.org/script/index.php

Пример (ImageMagic):
convert input.jpg -resize 200 output.jpg

Пример (sips), внимание, sips меняет исходную фотографию.
Используйте команду cp для того чтобы переместить фотографию из папки в папку
sips --resampleWidth 200 input.jpg
